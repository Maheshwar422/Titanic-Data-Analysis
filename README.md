# Titanic Data Analysis 🚢

## 📊 Project Overview
This project analyzes the Titanic dataset to understand survival patterns using Python.

## 🔧 Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📈 Key Insights
- Females had higher survival rate
- 1st class passengers survived more
- Age played a role in survival

## ▶️ How to Run
1. Open in Google Colab
2. Upload dataset
3. Run all cells
